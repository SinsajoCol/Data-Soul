# 02 Acceso a la aplicación

### Modalidad de acceso

La aplicación puede ejecutarse de dos formas:

* **Modo local:** Se necesita tener una cuenta de GitHub para acceder al repositorio fuente, y Git instalado para clonarlo.
* **Modo despliegue:** mediante cualquier navegador con acceso a internet.

### Requisitos del sistema

* Navegador actualizado (Google Chrome, Firefox, Edge...)
* Conexión a internet estable.&#x20;

### Instrucciones de acceso (modo local)

1. Clonar el repositorio del proyecto.
2. Acceder a la carpeta local dónde clonó el repositorio
3. Buscar el archivo `Index` y dale doble click.
4. Se abrirá una pestaña en tu navegador principal con una ruta como esta  `http://localhost:3000`.

### Instrucciones de acceso (modo despliegue)

1. Ir a la URL del servido**r (Adjuntar link de desaplique**
2. Accederás al la pantalla principal.
3. Elige la función que desees realizar.
